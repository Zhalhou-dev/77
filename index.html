About console


Visual Studio is a development environment that programmers use to create websites, web applications, web services, and mobile applications. It is a comprehensive integrated development environment (IDE) that allows users to write, edit, debug, and build code, and then deploy their app.



A console is a computer terminal that allows users to input commands and view output. It is often connected to a remote computer or computer system that is controlled from the console. It can be accessed by going Ctrl+shift+i in the web screen.To print on the console we use console.log() for javascript
